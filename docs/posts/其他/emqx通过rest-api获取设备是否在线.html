    emqx通过REST API获取设备是否在线

    2024-12-16 · 其他

    本文通过emqx REST API的方式来进行配置

    1. 创建一个API密钥

    1. 记录密钥的名称

    API key是用户名称，Secret KEY是密码

    1. 浏览器登录验证

    用户名就是 API Key，密码就是Secret KEY.

    登录上之后就可以获取到设备的信息了。

    可以参考： http://localhost:18083/api-docs/index.html 支持哪些API

    官网参考： https://docs.emqx.com/zh/emqx/latest/admin/api.html
